# KATCHCON Site

This site is ready to be published to GitHub Pages.

## Publish

1. Create a GitHub repository.
2. Run:
   - git init
   - git add .
   - git commit -m "Initial commit"
   - git branch -M main
   - git remote add origin https://github.com/<your-username>/<your-repo>.git
   - git push -u origin main
3. In GitHub, enable Pages and choose the main branch.

The site uses the Lato font via the Google Fonts import in css/style.css.
